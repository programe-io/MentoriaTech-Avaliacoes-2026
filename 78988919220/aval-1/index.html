Introdução à Web e Internet

A internet é uma rede global de computadores interconectados que permite a troca de informações entre dispositivos.

O que é a Web?

A Web é um serviço que funciona sobre a internet, permitindo acessar sites e páginas por navegadores.

Tecnologias principais

  • HTML - estrutura das páginas
  • CSS - estilo e design
  • JavaScript - interatividade
