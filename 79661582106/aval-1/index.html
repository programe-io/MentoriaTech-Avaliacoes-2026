FATOS QUE QUASE NINGUÉM CONHECI SOBRE O MINECRAFT

O ERRO "MINCERAFT" : Existe uma chance de 0,01% de que, ao abrir o jogo, o logo no menu principal exiba "Minceraft" em vez de Minecraft. MODO DEBUG ESCONDIDO : Ao criar um novo mundo, se você segurar a tecla Alt enquanto clica no botão "Tipo de Mundo", o jogo revelará o Modo Debug. Ele gera um mapa fixo contendo todos os blocos e estados de blocos existentes no código do jogo. LINGUA PIRATA : Digitar "excited ZE" no campo de busca do livro de receitas ou na bancada de trabalho altera todos os nomes de itens e blocos para o idioma Pirate Speak (Falar como Pirata). CRASH INSTANTÂNEO : Pressionar a combinação F3 + C por alguns segundos força o jogo a travar propositalmente. Isso é usado por desenvolvedores para gerar relatórios de erro, mas pode ser usado como pegadinha.
